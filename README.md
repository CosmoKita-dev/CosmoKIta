# CosmoKIta
ini adalah sourcecode dari web CosmoKIta yang mengikuti lomba FESTIKA jatim 2025

Cara Menjalankan Proyek:
a)	Ekstrak folder 'CosmoKita'.
b)	Buka file 'index.html' menggunakan browser.
c)	Aplikasi akan berjalan otomatis.
d)	Pastikan semua subfolder tetap berada pada struktur semula.
e)	Website bisa dibuka dari komputer maupun perangkat mobile
